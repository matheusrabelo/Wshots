# wshots
Website screenshots

## Install
```
$ npm install wshots -g
```

## Usage
Options

-w, --website [url]
    add website

-v, --viewport [widthxheight]
    add viewport

-d, --dom-element [query]
    query dom element

-h, --help
    show help info

## Example
```
$ wshots -w https://matheusrabelo.github.io/site/ -v 1024x768 -v 800x600 -d header
```
